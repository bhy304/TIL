## Set 집합

{1, 2, 3},{4, 5, 6}

합집합\(\|\), 교집합\(&\), 차집합\(^\)

union\(\)

```bash
>>> s1 = {1,2,3}
>>> s2 = {3,4,5}
>>> s1 | s2
{1, 2, 3, 4, 5}
>>> s1 & s2
{3}
>>> s1 ^ s2
{1, 2, 4, 5}
>>> s1
>>> s2
>>> s1.union(s2)
>>> ss = s1.union(s2)
>>> s1.update(s2)
{1, 2, 3, 4, 5}
```
